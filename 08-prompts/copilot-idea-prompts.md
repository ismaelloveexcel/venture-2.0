# Copilot Prompts — Idea Generation

These are pre-built prompts. Open Copilot Chat (Ctrl+Shift+I), pick one, paste it in.

---

## Score My Idea

```
Score this venture idea using my 8-criteria niche scorecard (1–10 each):
IDEA: [paste idea]
Criteria: Pain Intensity, Niche Specificity, Willingness to Pay, Reachability,
Speed to First $, Competitive Gap, AI Leverage, Personal Fit.
Give total/80. Verdict: GO / CONDITIONAL GO / NO-GO. Top risk in one sentence.
```

---

## Generate 10 Ideas for My Skills

```
My background: [paste skills, industries, tools you know]
Generate 10 productized AI service ideas I can start in 30 days.
Each: niche, pain, what I deliver, price, clients needed for $10k/month.
```

---

## Customer Interview Questions

```
I'm validating this idea: [paste idea]
Write 8 discovery interview questions to understand:
- How painful the problem really is
- What they're doing now to solve it
- What they'd pay to have it solved
- What they've already tried
Questions should feel conversational, not like a survey.
```

---

## Niche Down

```
Niche down this idea 3 levels to find the micro-niche with highest pain and willingness to pay:
IDEA: [paste idea]
Level 1 (broad) → Level 2 (specific) → Level 3 (micro).
For Level 3 write: one-line offer, price, 3 reasons this micro-niche would pay immediately.
```
