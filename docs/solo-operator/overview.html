When You Make Changes

One thing at a time. Here's what each change affects:

  • Message change: Updates personalization quality and expected response rates; requires regenerating all messages and resending.
  • Audience change: Updates prospect count, reach, and trend direction; changes the shortlist and all downstream data.
  • Timing change: Updates delivery schedule and response curve; may affect engagement metrics based on when prospects see messages.
  • Best practice: Change one variable per cycle so you can measure what actually worked.
