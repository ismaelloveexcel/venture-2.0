# Founder strategy note — wedge, proof, revenue (v1)

**Do not compete as a lead-gen vendor; compete as a tightly controlled outbound system for one buyer, one outcome, and one proof path.**

Win by being **narrower, more opinionated, and more provable** than generic “lead gen.” The crowded field is mostly **agency-style volume**, **AI-personalization theatre**, and **spam-first cadence**. You are not selling “more emails”; you are selling **discipline**: one ICP, caps, review, link-free first touch, and a reply path that earns the next step.

**Wedge (defensible):** one **buyer**, one **role**, one **problem**, one **motion**—with explicit guardrails your buyer already wishes someone enforced.

**Trust sequence (not a feature list):** cold email earns attention → micro-yes → walkthrough only after clear consent. That is the contrast to “book a call + seven links.”

**Proof (operational, internal first):** track one chain for each frozen cohort: same ICP → same message version → same deliverability rules → same reply taxonomy → **delivered → `walkthrough_yes` → conversations.** The ratio that matters early is **delivered-to-`walkthrough_yes` conversion**—not opens or raw replies alone. Keep this internal until you deliberately turn it into a public proof artifact.

**Revenue lane: pilot-first.** Target **2–4 paid pilots per month** at **$2.5k–5k** each to approach about **$10k in monthly revenue** before you chase low-ticket SaaS at scale. Treat **MRR** as a separate milestone: recurring revenue only after pilots convert into **retainers or subscriptions**, not from one-time pilot fees.

**Honest time anchor (90 days):** treat **$10k in the next 30 days** as realistic only with an existing pipeline; otherwise use **90 days** as the first serious checkpoint: enough delivered volume in one frozen cohort to read **strict** `walkthrough_yes` rate and call quality, then price and tighten ICP.

**The first goal is not scale, it is repeatability: can the same ICP, same message, and the same follow-up path produce consistent qualified replies?** If yes, raise price or narrow segment; if no, fix the chain before you scale sends.
