# Decision Log

**Rule: Log every major decision. If you're going in circles, check this file first.**

---

## How to Use
1. Every time you decide to pursue, pivot, or kill an idea — log it here.
2. Before starting a new idea, read the last 3 entries to avoid repeating mistakes.
3. Ask Copilot: "Review my decision log and tell me if I'm repeating a pattern."

---

## Log Template

```
## Decision #[N] — [Date]
**Idea/Topic:** 
**Decision Made:** [ ] Pursue  [ ] Pivot  [ ] Kill  [ ] Pause
**Reason:**
**What I tried first:**
**Key assumption that changed:**
**Next action:**
```
