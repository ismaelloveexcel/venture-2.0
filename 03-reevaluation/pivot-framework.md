# Pivot Framework

Use this when an idea isn't working but you don't want to quit prematurely.

---

## When to Pivot vs. When to Persist

Ask Copilot: "Based on these results [paste data], should I pivot or persist? Use the pivot framework."

### Persist if:
- You've sent fewer than 250 outreach messages
- You've had fewer than 5 real sales conversations
- You've been at it less than 4 weeks
- Prospects understand the offer but aren't ready yet

### Consider a pivot if:
- 50+ contacts and zero positive replies
- 5+ calls and zero interest even in a pilot
- Repeated objection you can't overcome
- You hate the work and can't imagine doing it daily

### Kill and restart if:
- The problem doesn't exist (no one cares)
- No one will pay what you need to charge
- You're fundamentally wrong about who the customer is

---

## Pivot Types (pick the smallest change first)

| Pivot Type | Description | Example |
|---|---|---|
| Niche pivot | Same offer, different customer | From dentists → chiropractors |
| Problem pivot | Same customer, different pain | From marketing → billing automation |
| Channel pivot | Same offer, different outreach | From cold email → LinkedIn DMs |
| Price pivot | Same offer, different packaging | From $2k/mo → $500 setup + $500/mo |
| Delivery pivot | Same outcome, different method | From done-for-you → done-with-you |

