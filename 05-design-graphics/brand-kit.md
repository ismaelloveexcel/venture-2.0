## Image & Icon Style

- Photography: Real people, candid — no stock-photo smiles
- Icons: Outlined, minimal (e.g. Heroicons, Phosphor)
- Illustrations: Flat, geometric if used

## AI Image Prompt Style Guide

For consistent AI-generated visuals, always include:

```
Style: clean, modern, minimal, professional
Colour palette: deep navy background, indigo accents, white text
Lighting: soft studio light, high contrast
Mood: confident, trustworthy, tech-forward
Format: 16:9 for hero images, 1:1 for social media
No watermarks, no text overlaid on image
```
